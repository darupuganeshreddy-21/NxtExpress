# 🚀 Flask REST API Web App (In Progress)

## 📌 Overview

This project is a simple web application built using **Flask** for the backend and **HTML, CSS, and JavaScript** for the frontend.

The application demonstrates full frontend–backend connectivity using **REST APIs** and JavaScript **fetch()** for handling HTTP requests.

> ⚠️ This project is currently in progress. Database integration, deployment, and additional enhancements are planned.

---

## 🛠️ Tech Stack

- **Backend:** Flask
- **Frontend:** HTML, CSS, JavaScript
- **API Communication:** REST APIs
- **Client-Side Requests:** JavaScript fetch()

---

## ✨ Features

- REST API implementation using Flask
- Support for all major HTTP methods:
  - GET
  - POST
  - PUT
  - DELETE

- Frontend connected to backend using JavaScript fetch()
- Clean and simple UI built with HTML & CSS
- Structured development with focus on teamwork, communication, and agile practices

## ⚙️ How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd project-folder
   ```

2. Create virtual environment:

   ```bash
   python -m venv venv
   ```

3. Activate it:
   - Windows:

     ```bash
     venv\Scripts\activate
     ```

   - Mac/Linux:

     ```bash
     source venv/bin/activate
     ```

4. Install Flask:

   ```bash
   pip install flask
   ```

5. Run the application:

   ```bash
   python app.py
   ```

6. Open in browser:

   ```
   http://127.0.0.1:5000/
   ```
